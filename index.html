Introduction to Graphic Design

Graphic design is all about creating visual content—like logos, posters, and website designs—that communicates a message. This course teaches you how to use design tools like Adobe Photoshop and Illustrator to create beautiful, eye-catching graphics. You’ll learn the basics of colors, fonts, and layout, so that you can make designs that not only look great but also convey the right message. By the end, you’ll be able to create designs that can be used for everything from websites to advertising.

Duration: 12weeks

In-person or online

Introduction to Product Design

Product design is the process of creating and developing new products that address specific user needs and deliver value in a functional, visually appealing, and user-friendly way. It involves a combination of creative, technical, and strategic skills to conceptualize, prototype, and refine products. Product design spans both physical and digital products, including everything from consumer goods and electronics to apps and websites.

Duration: 12weeks

In-person or online

Introduction to Data Science

Summary: Data science is like being a modern-day scientist who uses data to find answers to big questions. This course teaches you how to gather, clean, and analyze large sets of data using tools like Python and R. You’ll learn how to look for patterns, make predictions, and create models that help companies make smarter decisions. By the end, you'll be able to turn raw data into valuable insights that drive business success and innovation.

Duration: 12weeks

In-person or online

Introduction to Data Analysis

Data analysis is the process of collecting, cleaning, transforming, and interpreting data to extract meaningful insights, make informed decisions, and identify trends or patterns. It is a key element across various fields, including business, finance, healthcare, marketing, and research, enabling organizations and individuals to leverage data for strategic advantage.

Duration: 12weeks

In-person or online

Introduction to Business Analysis

Business analysis is like being a detective who solves problems for a company. In this course, you’ll learn how to understand a business’s needs, figure out what’s going wrong, and then help find solutions. You’ll learn how to gather information from different teams, analyze it, and suggest improvements that can help a company run more smoothly. Whether it’s improving processes, technology, or customer experiences, business analysis helps companies make better decisions and grow.

Duration: 12weeks

In-person or online

Introduction to Front-end Development

Front-end web development is about creating the parts of a website that users see and interact with. This course teaches you how to use HTML, CSS, and JavaScript to design and build the look and feel of a website, including colors, fonts, layouts, and interactive elements. By the end, you’ll be able to create user-friendly websites that look great and work well across different devices.

Duration: 12weeks

In-person or online

Introduction to Back-end Development

Back-end web development focuses on the “behind-the-scenes” parts of a website that users don’t see. This course teaches you how to work with servers, databases, and code that process information and make websites function. Using languages like Python, PHP, or Java, you’ll learn how to create the systems that manage data, handle requests, and keep everything running smoothly. By the end, you’ll know how to make websites work efficiently and securely.

Duration: 12weeks

In-person or online

Introduction to Mobile Development

Mobile development is about building apps for smartphones and tablets. In this course, you’ll learn how to create apps for Android and iOS using programming languages like Java, Swift, or Flutter. You’ll explore how to make apps that are user-friendly and perform well on mobile devices. By the end, you’ll be able to design and develop mobile apps that users can download from app stores.

Duration: 12weeks

Introduction to Product Marketing / Digital Marketing

You’ll learn strategies to position a product in the market, make it stand out, and drive sales. By the end, you’ll know how to create marketing campaigns that connect with customers and build brand loyalty.

Introduction to Project Management

Project management is about planning and overseeing projects to make sure they get done on time and within budget. This course teaches you how to set clear goals, organize tasks, and communicate with your team. You’ll learn methods like Agile and Waterfall to manage projects, ensuring that everything runs smoothly from start to finish. By the end, you’ll have the skills to lead projects, manage teams, and deliver successful results.

In-person or online

Introduction to Database Management and Data Governance

Database management is about organizing and storing information so it can be easily accessed and used. Data governance is about setting rules and standards for managing that data to keep it accurate, secure, and private. In this course, you’ll learn how to work with databases, use tools to organize data, and understand policies that protect it. By the end, you’ll know how to manage and protect data, which is essential for making good business decisions and ensuring data privacy.

Duration: 12weeks

In-person or online

Introduction to Cybersecurity

Cybersecurity is about protecting data and information from bad guys who try to steal it. This course teaches you how to secure computers, networks, and websites to prevent attacks like hacking, viruses, or data theft. You’ll learn how to identify weaknesses in security and use tools to protect against these risks. By the end, you’ll know how to keep sensitive information safe in an increasingly connected world.

Duration: 12weeks

In-person or online

Introduction to Cloud Computing

Cloud computing is about using the internet to store and access data instead of relying on your computer’s hard drive. In this course, you’ll learn how businesses use cloud services (like Google Cloud or AWS) to store information, run applications, and collaborate online. You’ll learn about the different cloud platforms and how to work with them to improve business efficiency, security, and flexibility. By the end, you'll know how cloud computing is changing the way companies operate and how you can use it to manage data and applications in the cloud.

Duration: 12weeks

In-person or online

Introduction to DevOps

DevOps is a way of working that brings together software development (Dev) and IT operations (Ops) to build, test, and release software faster and more reliably. This course teaches you how to set up automated processes that allow developers to quickly update software while keeping it stable and secure. You’ll learn tools like Jenkins, Docker, and Kubernetes to manage these processes. By the end, you’ll know how DevOps helps teams collaborate better and deliver software more efficiently.

Duration: 12weeks

In-person or online
